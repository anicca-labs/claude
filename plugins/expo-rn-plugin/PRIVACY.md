# Privacy Policy

Last updated: 2026-07-02

`expo-rn-plugin` is a Claude Code plugin that runs locally on your machine. It
does not collect, store, transmit, or share any personal data.

## What the plugin does

- Scaffolds code, reviews translations, and provides development skills that run
  entirely within your local Claude Code session.
- Ships two MCP servers (`expo-mcp-server`, `database-mcp-server`) that connect
  only to endpoints **you** configure — your own Expo project, Supabase,
  Firebase, or REST backend. Any credentials you provide stay on your machine
  and are sent only to those services you own.

## What the plugin does not do

- It contains no analytics, telemetry, tracking, or "phone-home" behavior.
- It sends no data to the plugin author or any third party.
